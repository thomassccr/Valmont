Installation en 3 étapes

1
Dézipe le téléchargement
Double-clique le fichier téléchargé : Valmont.app apparaît.
2
Glisse Valmont dans Applications
Ouvre le Finder et glisse Valmont.app dans le dossier Applications.
3
Premier lancement
Clic droit sur l'app → « Ouvrir » → « Ouvrir ». Si macOS insiste : Réglages Système → Confidentialité et sécurité → « Ouvrir quand même ». Une seule fois — ensuite elle s'ouvre normalement et se met à jour toute seule.
